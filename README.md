# A-Lightweight-Restoration-Assisted-Framework-for-Occlusion-Robust-Longan-Picking-Stem-Detection

stemPoint.py – performs restoration first, followed by keypoint detection.

stemSeg.py – performs restoration first, followed by segmentation.

stemrestore – is the program used for stem restoration.

img – contains test images captured under different scenarios.

model – includes the architecture files of OSRNet.

output – contains the resulting images from restoration and detection.

weight – stores the model weights used.

Model weights and more test images will be released after the paper is published.
